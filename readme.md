### 配列のspliceメソッドの使い方

	array.splice(削除開始位置、削除配列数、挿入要素)

	-削除開始位置：整数値を指定する。指定した配列番号から先が削除される。

	-削除配列数　：削除したい配列の個数を指定する。指定した個数分削除開始位置から削除される。

	-挿入要素　　：削除後に挿入したい要素。javascriptなので同じ配列に様々な型の要素を格納する事が出来る。コンマで区切って複数個指定できる。


**myArray.splice(3, 10, "test")　の場合**

まず、３番目の配列要素から１０個分の配列要素が削除される（つまり配列番号３〜12番が削除される）。

その後、"test"という配列要素が挿入される。



**一つの要素を削除したい場合**

myArray.splice(x, 1)とすると、　x番目の配列要素が削除される。

**削除せず単に挿入したい場合**

myArray.splice(x, 0, "insert", "insert2")として、　２番目の引数を０にすると何も削除されず、挿入のみ行われる。

**特定の配列番号以降を削除したい場合**

myArray.splice(3)とすると３番目の配列要素以降が全て削除される。
